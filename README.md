# 배달 어플 구현하기 

## 1단계 시중에 나와있는 배달 어플을 보면서 기능을 유추해보자.
### 사용자 입장
- 회원가입 
- 로그인/로그아웃
- 음식점 조회
- 음식 조회
- 음식선택(장바구니)
- 결제
- 결제 시 쿠폰 사용 여부
- 진행상황 보기
- 리뷰 작성
- 리뷰 조회 및 삭제
- 주문 내역 확인
- 음식점 찜하기

### 업주 입장
- 가게 등록
- 가게에서 팔 음식 등록
- 내가 등록한 음식 조회
- 내가 등록한 음식 수정, 삭제
- 주문 가능 시간 설정
- 주문 시 알림
- 쿠폰 등록 
- 리뷰 확인 및 답변 등록
- 답변 삭제
- 개발사 문의 등록(?)
